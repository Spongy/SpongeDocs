Installing SpongeForge¶

O SpongeForge integra-se no Minecraft Forge por iso também podes usar mods do Forge. Na verdade, o Sponge é ele próprio um mod para o Forge que depois carregas os plugins do Sponge, mas isto é um detalhe técnico.

Users who do not want to use Minecraft Forge can consider Installing SpongeVanilla.

Download¶

Check Download for further information.

Reading the Download Filename¶

Quando descarregares o SpongeForge, o nome do ficheiro vai conter algumas informações relevantes sobre versões. Inclui o número da build do Forge necessária para usar o SpongeForge. Outras versões do Forge que não a especificada, ainda que próximas, não funcionarão.

No entanto, o SpongeForge costuma ser atualizado rapidamente após cada lançamento do Forge, por isso, não terás de usar uma versão do Forge desatualizada para usares o SpongeForge.

O formato do nome do ficheiro é spongeforge-{MCVersion}-{ForgeVersion}-{SpongeAPIVersion}-{SpongeBuildId}

MCVersion The minecraft version. Only clients compatible with this version can connect.
ForgeVersion The version of Forge this file is built for. Preferably your server should run this exact version of Forge.
SpongeAPIVersion The version of the SpongeAPI implemented by this file. This is what Sponge plugins depend on.
SpongeBuildId The build number of Sponge. This is what you should supply when reporting bugs or seeking support.

Example¶

O ficheiro chamado spongeforge-1.8-1577-3.0.0-BETA-1000.jar é compatível com a versão do minecraft 1.8, requer a build 1.8-11.14.4.1577 do Forge, fornece a versão``3.0.0``da SpongeAPI e foi a ``1000``ª build do SpongeForge.

Nota

Os mods normais do Forge costumam correr em qualquer versão do Forge para uma dada versão do Minecraft (ex. 1.8) sem quaisquer problemas. No entanto, o SpongeForge precisa de acesso a, entre outras coisas, partes internas do Forge, em que a maior parte dos mods não deve mexer, muito menos modificar como o Sponge faz. Como os desenvolvedores do Forge têm liberdade para alterar o código interno sempre que querem, a garantia de compatibilidade com versões mais antigas não se verifica com o SpongeForge.

Installing SpongeForge¶

Nota

If you use (or are planning to use) a game server host, they may have a control panel that can install Sponge for you.

Aviso

When using the Mojang installer, Mojang makes use of their own Java version and not the one you installed on your system. The installer currently ships with Java 1.8.0_25 for Windows and 1.8.0_60 for OSX. Note that Sponge requires at least 1.8.0_40 or above to run properly. You can grab the Launcher without included Java here: official Minecraft Launcher

Single Player / In-Game LAN Servers¶

  1. Descarrega o instalador do Minecraft Forge do site do Minecraft Forge. Certifica-te de que usas exatamente a build com o mesmo número que a especificada no nome do ficheiro do Sponge descarregado.

  2. Run the provided Forge installer. A new Forge profile will be created in the Minecraft launcher.
  3. Open the Minecraft launcher, and select the new Forge profile.
  4. Click “Options” and click “Open Game Dir”.
  5. Download SpongeForge from the Sponge website and put it into the mods folder. Create the folder if it does not yet exist.
  6. Sponge should work in both in single player and if you open your world to LAN.

A seguir, aprende como podes configurar o Sponge e como podes gerir o teu servidor (incluindo a instalação de plugins).

Dedicated Servers¶

Nota

Se já tens um servidor Forge, basta pores o Sponge mod na pasta mods. Lembra-te de atualizares a versão do Forge para a versão específica requerida pelo Sponge.

Installing Forge via Commandline¶

  1. Visita o site do Minecraft Forge <http://files.minecraftforge.net/>`e clica em “Show all downloads” para veres uma lista completa das opções disponíveis. Identifica a versão que corresponde à indicada :doc:`no nome do ficheiro do SpongeForge que descarregaste <spongeforge>, e coloca o rato por cima do (i) ao lado de “installer” para teres o link de download direto.

  2. Use your favorite download method to download the jar to its destination. Example: wget http://url.to/forge-version-installer.jar
  3. From the folder in which you wish to install Forge, execute the jar with the --installServer option. Example: java -jar forge-version-installer.jar --installServer
  4. Continue to Adding SpongeForge to Forge below.

Installing Forge via GUI¶

  1. Descarrega o instalador do Minecraft Forge do site do Minecraft Forge <http://files.minecraftforge.net/>`que corresponde à versão mostrada no :doc:`nome do ficheiro do SpongeForge que descarregaste <spongeforge>.

  2. Run the provided Forge installer, select “Install Server”, choose an empty folder to place the server’s files, and then click OK.
  3. Continue to Adding SpongeForge to Forge below.

Adding SpongeForge to Forge¶

  1. Download SpongeForge from the Sponge website and put it into the mods folder in your server directory. Create the folder if it does not yet exist.
  2. You may now launch the server via command or launch script java -jar forge-version-XYZ.jar.

  3. Se estiveres a trabalhar a partir de casa, configura o Encaminhamento de portas para que os outros se possam juntar.

De seguida, aprende a criar e utilizar um launch-script, configurar o Sponge e gerir o teu servidor (incluindo a instalar plugins).
