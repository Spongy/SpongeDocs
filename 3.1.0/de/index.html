Sponge Dokumentation¶

Einführung¶

Willkommen in den SpongeDocs, der offiziellen Dokumentation des quelloffenen Sponge-Projekts.

Die SpongeAPI ist deine brandneue Plugin API für Minecraft. Du kannst sie auf einem Vanilla Server (als SpongeVanilla) oder auf deinem gemoddeten MinecraftForge Server (als SpongeForge) nutzen. Lade es hier herunter.

Bist du dir nicht sicher was du suchst? Probier unsere Über das Sponge Projekt Sektion, die eine kurze Einführung enthält, unsere Häufig gestellte Fragen und den Artikel über die The Structure of the Sponge Project.

Serverbetreiber sollten unsere Tipps und Hilfestellungen im Einen Server erstellen Abschnitt lesen. Dort lernst du wie man einen Sponge basierten Server installiert, betreibt und wartet.

Das Sponge Projekt hat mehrere Hauptkomponenten. Besuche die GitHub Repositories oder erfahre mehr über sie hier: SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | Mixin | Ore

Bemerkung

Sponge befindet sich immernoch in der Entwicklung. Beta-Versionen von SpongeForge und Entwicklungsversionen von SpongeVanilla können hier heruntergeladen werden: Download. Eine offizielle Release-Version ist derzeit noch nicht verfügbar.
