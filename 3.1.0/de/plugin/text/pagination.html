Der Pagination-Service¶

Tipp

For a basic understanding of services, make sure you read Arbeiten mit Services first.

Der PaginationService ist dazu da, um Inhalt auf mehrere eigenständige Seiten aufzuteilen. Der Service stellt den PaginationBuilder zur Verfügung, mit dem man Attribute wie Titel, Inhalt, Kopfzeile und Padding festlegen kann.

Mit dem PaginationBuilder arbeiten¶

Zuallererst rufen wir eine Instanz des PaginationServices ab und erstellen einen neuen PaginationBuilder:

import org.spongepowered.api.Sponge;
import org.spongepowered.api.service.pagination.PaginationBuilder;
import org.spongepowered.api.service.pagination.PaginationService;

PaginationService paginationService = Sponge.getServiceManager().provide(PaginationService.class).get();
PaginationBuilder builder = paginationService.builder();

Es gibt zwei verschiedene Wege zum Festlegen der Inhalte einer Pagination:

  • Mit einer Iterable<Text>

import org.spongepowered.api.text.Text;

import java.util.ArrayList;
import java.util.List;

List<Text> contents = new ArrayList<>();
contents.add(Text.of("Item 1"));
contents.add(Text.of("Item 2"));
contents.add(Text.of("Item 3"));

builder.contents(contents);

Bemerkung

If the Iterable is a List, then bidirectional navigation is supported. Otherwise, only forwards navigation is supported.

  • Mit einem Array von Texten

builder.contents(Text.of("Item 1"), Text.of("Item 2"), Text.of("Item 3"));

You can also specify various other components of a paginated list, such as a title, header, footer, and padding. The diagram below shows which component is displayed in each part of the paginated list. In the following diagram, the padding string is shown as the letter p.

pppppppppppppppppppppppp Title pppppppppppppppppppppppp
Header
Item 1
Item 2
Item 3
...
ppppppppppppppppppppppp < 2/3 > ppppppppppppppppppppppp
Footer

To achieve the preceding output, we might use the following builder pattern:

builder.title(Text.of("Title"))
    .contents(Text.of("Item 1"), Text.of("Item 2"), Text.of("Item 3"))
    .header(Text.of("Header"))
    .footer(Text.of("Footer"))
    .paddingString("p");

Bemerkung

Mit Ausnahme der Inhalte sind alle Komponenten der nummerierten Liste optional. Ein Titel wird jedoch dringend empfohlen

Um die nummerierte Liste abschließend an eine CommandSource zu senden, nutze PaginationBuilder#sendTo(CommandSource source).

Und das war’s! Zur Erinnerung: Eine voll funktionsfähige nummerierte Liste kann generiert und an eine zuvor definierte cmdSource gesendet werden mithilfe des folgenden Codes:

PaginationService paginationService = Sponge.getServiceManager().provide(PaginationService.class).get();

paginationService.builder()
    .title(Text.of("Title"))
    .contents(Text.of("Item 1"), Text.of("Item 2"), Text.of("Item 3"))
    .header(Text.of("Header"))
    .footer(Text.of("Footer"))
    .paddingString("p")
    .sendTo(cmdSource);
