Berechtigungen verwalten¶

Du kannst einstellen wer Zugang zu was hast, wenn du einen Server betreibst der Zugangsberechtigungen (engl. permissions) nutzt. Die verfügbaren Berechtigungen für Minecraft und Sponge fondest du Befehle.

Operator-Berechtigung¶

Minecraft bietet von Hause aus einen einfachen Weg des Permission Management (Rechteverwaltung): Es erlaubt Spielern den Status “Operator” (kurz “op”; Moderator) zuzuweisen. Allgemeine Informationen über den Op Status können hier nachgelesen werden: http://minecraft.gamepedia.com/Op

Die Berechtigungen des Op Status können durch Ändern des op-permission-level in der server.properties Datei angepasst werden.

Eine Liste der standardmäßig verfügbaren Minecraft Server Befehle, die Spieler mit Op Rechten nutzen können, ist hier verfügbar: Minecraft Wiki

Warnung

Minecraft hat keine exakte Rechteverwaltung, nur OP. Dies ist ein sehr hohes Permission-Level und sollte nur Spielern zu Verfügung gestellt werden denen man vertraut. Um einzelne Permissions zu setzen benötigt man ein Permissions Plugin oder eine entsprechende Mod. Sponge ist kein Plugin um Permissions zu verwalten.

Bemerkung

Manche Plugins oder Mods können OPs auch bestimmte Rechte zuweisen.
