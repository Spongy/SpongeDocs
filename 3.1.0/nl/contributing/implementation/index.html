Developing Sponge¶

This sections describes the best way to approach the project when you’re intending to develop Sponge itself. It is mandatory to be familiar with git and our Code Style. Basic knowledge of the The Structure of the Sponge Project is also very helpful.

If you want to join us in creating Sponge, head straight over to the Submitting a Pull-Request page.
