SpongeVanilla installeren¶

SpongeVanilla is een vanilla implementatie van de SpongeAPI als een zelfstandige server.

Overzicht¶

SpongeVanilla is een implementatie van de SpongeAPI die gemaakt is door het patchen van de vanilla Minecraft server. Dit betekent dat het een zelfstandige server is en maakt dus niet gebruik van Minecraft Forge of Forge Mod Loader (FML). SpongeVanilla wordt parallel ontwikkeld aan de Forge versies van Sponge, als een alternatief voor gebruikers die niet een Forge server willen. Oorspronkelijk begonnen als een onafhankelijk project genaamt Granite, door ontwikkelaars AzureusNation en VoltaSalt, het SpongeVanilla team trad officiel toe tot het Sponge ontwikkel team in maart 2015.

Download¶

Check Download voor meer informatie.

SpongeVanilla installeren¶

SpongeVanilla werkt alleens als zelfstandige server.

  1. Download de SpongeVanilla .jar van de Sponge website.

  2. Voer het uit via de command line: java -jar spongevanilla-watdanook.jar

  3. Set up Poort Doorsturen (Port Forwarding) om ervoor te zorgen dat anderen kunnen verbinden.

Waarschuwing

Niet dubbelklikken op het .jar bestand!
