Implémentation de l’API Economie¶

Sponge fournit des interfaces pour les plugins d’Economie afin d’implémenter et de créer une API. Pour construire une API d’Economie complete, vous devez implementer six classes:

  • Currency
  • EconomyService
  • TransactionResult
  • TransferResult
  • UniqueAccount
  • VirtualAccount

Choses à considérer lors de l’implementation de l’API Economie¶

  • Est-ce que je veux supporter plusieurs types de monnaie?

  • Est-ce que je veux que mon plugin exécute des actions différentes en fonction du contexte?

Implémenter l’API Economie est relativement simple et le JavaDocs peut être d’une grande aide si vous êtes incertain par rapport aux rôles des méthodes.
