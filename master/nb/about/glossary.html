Sponge-ordliste¶

Dette er en guide til de mest vanlige reserverte ord, fraser og forkortelser som brukes i Sponge-prosjektet. I noen tilfeller legges det ved lenker til mer informasjon. Dette er ikke ment som en guide til Sponge-API-en eller Java.

Obs

Ord defineres i sammenheng med Minecraft og/eller Sponge-prosjektet som nødvendig.

Definisjoner¶

API

Programmeringsgrensesnitt - en måte å gjøre modifikasjon enklere.

Bidragsyter

En som bidrar med kode til SpongePowered-prosjekter.

Kjernemod

En Forge-mod som lastes inn før Mojang-klassene lastes. Formålet med en kjernemod er å endre på eller overskrive faktisk Minecraft-kode. http://www.minecraftforge.net/wiki/Core_Mod

Crowdin

Et nettbasert oversettelsessystem for dokumentasjon. SpongeDocs bruker det til å oversette dokumentasjonen til mange forskjellige språk. https://translate.spongepowered.org/project/sponge-docs

Developer

En som programmerer kode (Sponge, plugins, mods, ...).

FLARD

Grønt, spiselig og høyeksplosivt. La det være og det lar deg være.

Forge

En langvarig Minecraft-modifikasjons-API, med sitt eget installasjonsprogram. Forge gjør det mulig med mods i enspillermodus, på lokalt nett (LAN) og på dedikerte servere. http://www.minecraftforge.net/wiki/

GitHub

Et gratis, offentlig, nettbasert system for å dele kode og opprette prosjekter. https://github.com/

Implementasjon

En mod eller et program som gjør at du kan bruke Sponge-API i Minecraft.

IRC

Internet Relay Chat, brukes som en chatplattform. Tilbyr chatrom for alt som har med Sponge å gjøre.

Issue

En måte å merke problemer, bugs, forbedringer og ting som må gjøres på GitHub og annensteds. Sponge-prosjektet bruker GitHub til behandling av Issues.

Java

Programmeringsspråket som både Minecraft og Sponge er skrevet i, og som kreves for å kjøre dem. http://java.com/en/

MCP

The Mod Coder Pack hjelper utviklere til å lage modifikasjoner til både Minecraft serveren og klienten. http://www.modcoderpack.com/website

Mixins

Spesialiserte programkomponenter som injiserer Sponge i Minecraft https://github.com/SpongePowered/Mixin/wiki

Mod

En Minecraft-modifikasjon som på en eller annen måte endrer spillet. Mods som er skrevet med bruk av Forge-API-en krever Forge for å virke, men noen mods kan installeres utenom.

NPC

Ikkespillbar karakter. Ethvert vesen som ikke blir kontrollert av en spiller, for eksempel en villager.

Ore

Den offisielle Sponge hosting siden for moduler, ikke ferdig ennå. Bruk Sponge- forumene i stedet.

Patreon

En tjeneste som tillater donasjoner til prosjekter og organisasjoner. Dersom du vil støtte Sponge, vil dette være rette sted! Sponge Patreon donasjons-side

Modul
A Minecraft mod that extends and changes Minecraft via the SpongeAPI, usable only on Sponge servers.
Prosjektledere
The folks in charge of the entire Sponge Project, responsible for co-ordinating the activities of the various Teams and sub-projects.
Pull Request (PR)

En metode for å legge bidragsytere til et åpent utviklingsprosjekt, slik som Sponge sitt GitHub lagringssted.

Sponge

Navnet på hele prosjektet.

SpongeAPI
The Minecraft Plugin API developed by the Sponge project.
SpongeCommon
A repository of code common to the official SpongeAPI implementations, used in building them. SpongeVanilla and SpongeForge use this repository.
SpongeForge
The implementation of SpongeAPI as a Forge Coremod.
SpongeDocs
Dedicated and frequently updated documentation, the best source of information on Sponge.
Sponge Forums
Home to the Sponge Project and the second best source of information on Sponge. Update announcements are posted regularly. https://forums.spongepowered.org/
Sponge stiftelsen
The financial side of the project, a separate entity that accepts, manages and distributes donations. http://spongefoundation.org/
SpongeVanilla
An implementation of Sponge API that runs on Vanilla Minecraft (and not Forge).
Spongie
The Sponge Mascot, a valiant little Sponge with a red cape.
Spongy
The sponge GitHub Bot that makes PR previews for the Docs.
Lagledere
The people who directs the Teams (WebDev, Docs, SysOps, subreddit, IRC).
