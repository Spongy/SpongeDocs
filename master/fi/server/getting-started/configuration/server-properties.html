server.properties¶

Oletustiedosto¶

Tässä on muokkaamaton 1.8.1 Minecraft palvelimen asetustiedosto.

#Minecraft server properties
#(File modification datestamp)
spawn-protection=16
max-tick-time=60000
generator-settings=
force-gamemode=false
allow-nether=true
gamemode=0
enable-query=false
player-idle-timeout=0
difficulty=1
spawn-monsters=true
op-permission-level=4
resource-pack-hash=
announce-player-achievements=true
pvp=true
snooper-enabled=true
level-type=DEFAULT
hardcore=false
enable-command-block=false
max-players=20
network-compression-threshold=256
max-world-size=29999984
server-port=25565
server-ip=
spawn-npcs=true
allow-flight=false
level-name=world
view-distance=10
resource-pack=
spawn-animals=true
white-list=false
generate-structures=true
online-mode=true
max-build-height=256
level-seed=
use-native-transport=true
motd=A Minecraft Server
enable-rcon=false

Asetuksen selitys¶

Kiitokset selityksille kuuluu Minecraft Wiki tekijöille.

Totuusarvosuodatin arvoilla on vain kaksi sallittua arvoa: true ja false. Lukujono arvo on aina kokonaisnumero. Jos alue on määritelty, täytyy arvon olla määritellyn alueen sisällä. Merkkijono arvo voi olla mikä tahansa symboli tai teksti.

Avain

Tyyppi

Oletusarvo

Kuvaus

Salli lentäminen

totuusarvosuodatin

 false

Sallii käyttäjien lentää palvelimella kun he ovat Survival tilassa, jos palvelimella on lisäosa joka sallii lentämisen.

Jos sallitset lentämisen, saattavat pahantekijät olla yleisempiä, koska se tekee heidän työstään helpompaa. Luovassa tilassa tällä ei ole vaikutusta.

false - Lento ei ole sallittua (pelaajat jotka ovat yli 5 sekuntia ilmassa, potkitaan ulos).

true - Lento on sallittu
allow-nether

totuusarvosuodatin

 true

Sallii pelaajien matkustaa Ala-Maailmaan.

false - Alamaailman portaalit eivät toimi.

true - Palvelin sallii Ala maailman portaalien luonnin ja niistä kulkemisen.
announce-player-achievements

totuusarvosuodatin

 true

Sallii palvelimen julkaista pelaajien saavutukset.

vaikeustaso

numeroarvo (0-3)

 1

Säätää vaikeustason (kuten monstereiden vahingon, nälän ja myrkkyjen vaikutuksen pelaajiin) palvelimella.

0 - Rauhallinen

1 - Helppo

2 - Normaali

3 - Vaikea
enable-query

totuusarvosuodatin

 false

Sallii GameSpy4 protokolla palvelimen. Tätä käytetään palvelimen infon hankkimiseksi.
enable-rcon

totuusarvosuodatin

 false

Sallii etäyhteyden palvelimen konsoliin.
enable-command-block

totuusarvosuodatin

 false

Sallii komentopalikat.
force-gamemode

totuusarvosuodatin

 false

Pakottaa pelaajien pelimuodon liittyessä palvelimeen.

false - Pelaajat liittyvät samassa pelimuodossa missä he lähtivät.

true - Pelaajat liittyvät aina oletuspelimuodossa.
gamemode

numeroarvo (0-3)

 0

Määrittelee palvelimen pelimuodon.

0 - Selviytyminen

1 - Luova

2 - Seikkailu

3 - Katsoja
generate-structures

totuusarvosuodatin

 true

Määrittelee luodaanko uusille alueille rakennuksia (kuten kyliä).

false - Rakennuksia ei luoda.

true - Rakennuksia luodaan.

Huomautus: Tyrmiä luodaan vaikka asetus olisi false.
generator-settings

Merkkijono

tyhjä

Asetukset joilla voidaan vaikuttaa maailmojen luontiin. Katso Superlitteä ja Kustomoitu Minecraft wikistä (ulkoiset linkit) jos haluat tietää mahdolliset asetukset.
hardcore

totuusarvosuodatin

 false

Jos asetat tämän true pelaajat poistetaan palvelimelta lopullisesti jos he kuolevat.
level-name

Merkkijono

 world

“level-name” arvoa käytetään maailman nimeämiseksi. Voit myös kopioida valmiin maailman tänne, ja nimetä tämä samaksi kuin kopioitu maailma jos haluat pelata siinä.

Erikoismerkkeihin kuten ‘ (kenoviiva) täytyy mahdollisesti lisätä eteen toinen kenoviiva (\).
level-seed

Merkkijono

tyhjä

Lisää seedi maailmaasi, samalla tavalla kuin yksinpelissä.

Joitakin esimerkkejä: minecraft, 404, 1a2b3c...
level-type

Merkkijono

 DEFAULT

Määrittelee luotavan kartan tyypin.

DEFAULT - Perusmaailma vuorilla, laaksoilla, vedellä, tms...

FLAT - Litteä maailma ilman mitään erikoisuuksia, tarkoitettu rakenteluun.

LARGEBIOMES - Sama kuin default, mutta kaikki biomit ovat suurempia.

AMPLIFIED - Sama kuin default, mutta maailmageneraattorin korkeusrajat on nostettu.

CUSTOMIZED - Sama kuin default, ellei generator-settings asetusta ole muutettu.
max-build-height

numeroarvo

 256

Maksimikorkeus jossa saa rakentaa. Maasto saattaa mennä tämän rajan yli jos se on alhainen.
max-players

numeroarvo (0- 2147483647)

 20

Yhtäaikaisten pelaajien maksimimäärä. Huomaa että mitä isompi määrä pelaajia, sitä tehokkaampi palvelin pitää olla. Huomaa myös että OP-pelaajia ei lasketa määrään mukaan, mutta ominaisuus ei tällä hetkellä toimi. Äärimmäisen suuret arvot saattavat rikkoa pelaajien oman pelin.
max-tick-time

numeroarvo (0- (2^63-1))

 60000

Maksimi aika millisekunteina, kuinka kauan yksi tick saa kestää, ennen kuin monitoriprosessi lopettaa palvelimen seuraavalla viestillä: “A single server tick took 60.00 seconds (should be max 0.05); Considering it to be crashed, server will forcibly shutdown”. Kun tämä yhtälö on tosi, annetaan palvelimelle System.exit(1) komento.

-1 - poista monitoriprosessi käytöstä
max-world-size

numeroarvo (1- 29999984)

 29999984

Tämä asettaa maailman maksimikoon palikoina, ilmaistuna säteenä. Maailman rajojen asettaminen tällä käskyllä onnistuu oikein, mutta raja ei oikeasti liiku yli tämän koon. max-world-size asettaminen suuremmaksi kuin oletusarvo, ei näytä tekevän mitään.

Esimerkkejä:

  • max-world-size asettaminen 1000 sallii 2000x2000 maailman.

  • max-world-size asettaminen 4000 antaa sinulle 8000x8000 maailman.
motd

Merkkijono

Tämä on Minecraft Palvelin

Tämä on viesti joka näkyy palvelinlistauksessa, palvelimen nimen alla.

  • Päivän viesti sallii värikoodien ja muokkauskoodien käyttämisen.

  • Jos päivän viesti on yli 59 merkkiä, palvelinlistaus luultavasti ilmoittaa yhteysvirheen.
network-compression-threshold

numeroarvo

 256

Oletuksena sallii pakettit joiden koko on n-1 bittiä, mutta suuremmat kuin n bittiä pakataan pienemmäksi. Pienempi arvo lisää pakettien pakkausta, mutta pienten pakettien pakkaus saattaa johtaa isompiin lopullisiin paketteihin.

-1 . poista pakkaus kokonaan käytöstä

0 - pakkaa kaikki paketit

Huomaa Ethernetin spesifikaatio vaatii että alle 64 bitin paketit pakataan 64 bitin kokoisiksi. Joten, arvon asettaminen alle 64 bittiin ei ole hyödyllistä. Ei ole myöskään suositeltavaa ylittää pakettien maksimikokoa (MTU - Maximum Transmission Unit), yleensä 1500 bittiä.
online-mode

totuusarvosuodatin

 true

Palvelin tarkistaa yhdistävän pelaajan tilin Minecraftin pelaajatietokannasta. Aseta tämä false vain jos palvelimesi ei ole yhteydessä internettiin. Väärennetyillä tileillä yhdistävät hakkerit voivat yhdistää palvelimeen jos arvo on false! Jos minecraft.net on kaatunut tai siihen ei saada yhteyttä, pelaajat eivät voi yhdistää palvelimeen jos arvo on true. Arvon asettaminen tarkoituksella falseksi kutsutaan “kräkkäämiseksi”, näin pelaajat joilla ei ole ostettua Minecraftia voivat liittyä palvelimelle.

false - Pois käytöstä. Palvelin ei yritä tarkistaa yhdistävien pelaajien nimiä.

true - Päällä. Palvelin olettaa olevansa yhdistettynä internettiin ja tarkistaa jokaisen yhdistävän pelaajan.
op-permission-level

numeroarvo (1-4)

 4

Asettaa admineiden oikeustason. Jokainen taso sisältää myös alemman tason oikeudet.

1 - Adminit voivat ohittaa syntymäalueen suojauksen.

2 - Adminit voivat käyttää /clear, /difficulty, /effect, /gamemode, /gamerule, /give, ja /tp, ja muokata komentopalikoita.

3 - Adminit voivat käyttää /ban, /deop, /kick, ja /op.

4 - Adminit voivat käyttää /stop.
player-idle-timeout

numeroarvo

 0

Jos arvo on muu kuin 0, pelaajat poistetaan palvelimelta jos he eivät ole aktiivisia määritellyn ajan sisällä.

Huomautus Aktiivisuusmittari resetoituu jos palvelin saa jonkun näistä paketeista:

  • 102 (0x66) WindowClick
  • 108 (0x6c) ButtonClick
  • 130 (0x82) UpdateSign
  • 14 (0xe) BlockDig
  • 15 (0xf) Place
  • 16 (0x10) BlockItemSwitch
  • 18 (0x12) ArmAnimation
  • 19 (0x13) EntityAction
  • 205 (0xcd) ClientCommand
  • 3 (0x3) Chat
  • 7 (0x7) UseEntity
pvp

totuusarvosuodatin

 true

Sallii PvP:n palvelimella. Pelaajat jotka ampuvat itseään nuolilla saavat vahinkoa vain jos PvP on sallittu.

false - Pelaajat eivät voi tappaa toisia pelaajia (tunnettu myös nimellä Player versus Environment (PvE))

true - Pelaajat voivat tappaa toisiaan.

Huomaa Ei suoraan pelaajien aiheuttama vahinko (kuten laava, tuli, TNT ja joissakin määrin vesi, hiekka ja sora) tekevät silti vahinkoa muihin pelaajiin.
query.port

numerarvo (1- 65534)

 25565

Määrittelee etäpalvelimen portin (katso enable-query)
rcon.password

Merkkijono

tyhjä

Asettaa etäkonsolin salasanan.
rcon.port

numerarvo (1- 65534)

 25575

Asettaa etäkonsolin portin.
resource-pack

Merkkijono

tyhjä

Linkki palvelimen omaan tekstuuripakettiin. Pelaaja voi valita käyttääkö sitä.
resource-pack-hash

Merkkijono

tyhjä

Lisä SHA-1 merkkijono resurssipaketista, pienillä heksadesimaali kirjaimilla. On suositeltavaa asettaa tämä. Tätä ei vielä käytetä resurssipaketin tarkistamiseen, mutta se auttaa pelaajien paikallisten väliaikaistiedoston varmennuksessa.
server-ip

Merkkijono

tyhjä

Aseta tämä jos haluat palvelimen kiinnittyvän johonkin tiettyyn IP-osoitteeseen. On hyvin suositeltavaa jättää tämä tyhjäksi.
server-port

numerarvo (1- 65534)

 25565

Vaihtaa portin jota palvelin kuuntelee (listen). Tämä portti pitää olla ohjattu palvelimeen jos palvelin on NAT-verkossa (jos sinulla on koti reititin/palomuuri).
snooper-enabled

totuusarvosuodatin

 true

Asettaa lähettääkö palvelin meta-dataa tasaisin väliajoin http://snoop.minecraft.net osoitteeseen. (ulkoinen linkki)

false - Poista datan lähetys käytöstä.

true - Salli datan lähetys.
spawn-animals

totuusarvosuodatin

 true

Määrittää sallitaanko eläimien syntyminen.

false - Kaikki eläimet katoavat heti, ja yhtäkään uutta ei synny.

true - Eläimet syntyvät normaalisti.

Vinkki: jos palvelimesi on hidas, aseta tämä false.
spawn-monsters

totuusarvosuodatin

 true

Määrittelee syntyykö monstereita.

false - Kaikki monsterit katoavat heti, eikä uusia enää synny, päivällä tai yöllä.

true - Monsterit syntyvät normaalisti; pimeässä sekä yöllä.

Tällä asetuksella ei ole merkitystä jos palvelimen vaikeustaso on Rauhallinen.

Vinkki: jos palvelimesi on hidas, aseta tämä false.
spawn-npcs

totuusarvosuodatin

 true

Määrittelee syntyykö kyläläiset.

true - Käytössä. Kyläläiset syntyvät.

false - Pois käytöstä. Ei kyläläisiä.
spawn-protection

numeroarvo

 16

Määrittelee syntymäpaikan alueen suojauksen. Tämän asettaminen 0 ei poista alueen suojausta. 0 suojaa yhden palikan alueella. 1 suojaa 3x3 alueen jonka keskipiste on syntymäpaikassa. 2 suojaa 5x5, 3 suojaa 7x7, jne. Tämä asetus ei tule esiin palvelimen käynnistyksessä, vaan vasta kun ensimmäinen pelaaja liittyy palvelimelle. Jos palvelimella ei ole Admineita, alueen suojaus poistuu automaattisesti käytöstä.
use-native-transport

totuusarvosuodatin

 true

Linux palvelimen optimointia: optimoitu pakettien lähetys ja vastaanotto Linuxissa.

false - Pois käytöstä. Poista Linuxin pakettien lähetyksen ja vastaanoton optimointi.

true - Päällä. Optimoi Linuxin pakettien lähetys ja vastaanotto.
view-distance

numeroarvo

(3-15)

 10

Asettaa pelaajalle lähetettävien alueiden määrän, mitattuna chunkkeina, joka suuntaan pelaajasta (alue, ei halkaisija). Tämä määrittää palvelimen oman katseluetäisyyden.

10 on suositeltava perusarvo. Jos palvelimesi on todella hidas, pienennä tätä arvoa.
white-list

totuusarvosuodatin

 false

Enables a whitelist on the server. See Sallittu-listan ylläpito. With a whitelist enabled, users not on the whitelist will be unable to connect. Intended for private servers, such as those for real-life friends or strangers carefully selected via an application process, for example.

false - Sallittujen lista ei ole käytössä.

true - Tiedostoa “whitelist.json” käytetään sallittujen listan tekemiseen.

Huomaa: Adminit lisätään automaattisesti listaan, heitä ei tarvitse erikseen lisätä sallittujen listaan.
