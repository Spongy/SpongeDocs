Ongelmanratkaisu¶

You’re probably here because something went wrong with your Sponge server. Let’s see if we can figure out what it was, and what to do about it.

Mahdolliset ongelmanlähteet

  • Java ei ole asennettu tietokoneellesi
  • Ongelma verkkoyhteydessä (tai mahdollinen DDoS-hyökkäys)
  • Liian vähän vapaata muistia
  • Rikkoutunut asetustiedosto (esim. editointivirhe)
  • Lisäosa (tai Modi) rikkoutunut
  • Käyttöjärjestelmä on epävakaa (esim. virus)
  • Korruptoitunut data
  • Ongelma näppäimistön ja tuolin välillä
  • Spongessa on bugi
  • Universumissa on jotakin vikaa

Java ei ole asennettu tietokoneellesi¶

Solution: Get Java. Visit the Installing Java for more information.

Ongelma verkkoyhteydessä (tai mahdollinen DDoS-hyökkäys)¶

Ongelmat: Verkkoyhteys on hyvin epävakaa, katkeilee, tai ei toimi ollenkaan.

Ratkaisu: Tarkista yhteytesi reitittimeesi tai modeemiisi. Katso onko selaimessasi samankaltaisia ongelmia. Voit käyttää ilmaista palvelua kuten speedtest.net tarkistaaksesi yhteytesi nopeuden. Ongelmana saattavat olla myös muut palvelut verkossasi tai tietokoneellasi. Tarkista että olet tehnyt tarvittavat porttiohjaukset reitittimeesi tai modeemiisi. Joskin epätodennäiköinen, saattaa ongelma olla myös DDoS-hyökkäys. Jos uskot että syy on tämä, ota yhteyttä palveluntarjoajaasi ja kysy heiltä neuvoja.

Liian vähän vapaata muistia¶

Ongelmat: Palvelin kaatuu, lisäviestinä “Out of Memory”.

Ratkaisu: Aseta tietokoneelle lisää minimimuistia lisäämällä aloituskomentoon -XX:MaxPermSize=128. Laajenna palvelimesi aloitus- ja maksimimuistimäärää (jos mahdollista) komennoilla -Xms1024M (1GB aloitusmuistia) ja -Xmx2048M (2GB maksimimuistia). Tarkkaile tietokoneesi vapaan muistin määrää nähdäksesi mitkä prosessit vievät muistia. Joudut ehkä lopettamaan jumiutuneita java-prosesseja tai käynnistämään tietokoneesi uudestaan. Muistivuodot saattavat johtua bugeista lisäosissa, joten niiden etsimiseen kuluu usein paljon aikaa.

Still an issue?: If you are still having issues despite the above and cannot increase the Heap Size, check in your Task Manager to see if you are using all available Memory. If you are, the only solution is to add more RAM to your system. If there is still plenty of memory available, you are running 32-bit Java. If you are using 32-bit Java, we recommend an upgrade to 64-bit Java, provided that your Operating System is also 64-bit.

Rikkoutunut asetustiedosto (esim. editointivirhe)¶

Ongelmat: Yksi (tai enemmän) lisäosa ei suostu latautumaan tai käyttäytyy oudosti. Palvelimen lokitiedosto sisältää viestejä lukuvirheistä käynnistyksen aikana. Palvelin saattaa kaatua ja tiedostot korruptoitua.

Solution: Stop the server, and check your edited files. Load backup files of any corrupted data. You may need to delete a config file entirely and allow it to regenerate upon server startup.

Lisäosa (tai Modi) rikkoutunut¶

Ongelma: Tämä saattaa olla mitä tahansa - mitä tahansa lisäosa tekeekään + X-mahdollisuus. Yleensä palvelin kaatuu ja logietiedosto täyttyy lisosan virheilmoituksista.

Solution: Stop the server, and check to see nothing has been corrupted. Be sure to check that it isn’t from an incorrectly edited config file (above). Remove suspect plugins and add them again one by one, restarting the server each time. The problem may originate from one plugin that is out of date - check for updates. Plugin conflict may also be the cause, having two incompatible plugins.

Käyttöjärjestelmä on epävakaa (esim. virus)¶

Ongelma: Palvelin kaatuu tai on tavoittamattomissa, ja muut osat käyttöjärjestelmästäsi käyttäytyvät oudosti.

Ratkaisu: Pysäytä kaikki. Katso tietokoneesi tarkasti läpi vistusten ja haittaohjelmien varalta. Hyviä työkaluja tähän ovat esimerkiksi AdwCleaner, Junkware Removal Tool, MalwareBytes ja yleisimmät antivirusohjelmat. Katso että palvelimen asetustiedostot ovat toimivat. Tarkista myös mahdolliset fyysiset viat, esimerkiksi viallinen virtalähde.

Korruptoitunut data¶

Ongelma: Maailmatiedostot eivät lataudu tai saavat palvelimen kaatumaan kun pelaaja menee tietyille alueille. Tietokannan korruptoituminen.

Ratkaisu: Lataa korruptoituneen datan varmuuskopio. Rikkoutuneita maailmoja varten on olemassa sovelluksia joilla niitä voi korjata. Tutki korruption mahdollista alkulähdettä - vääränlainen lisäosa, tietokannan ajuri, yhtäkkinen virrankatkaisu tai jotakin muuta? Varmista aina että tärkeä data on varmuuskopioitu turvalliseen paikkaan.

Ongelma näppäimistön ja tuolin välillä¶

Ongelma: Kaikki toimi hyvin eilen. Tänään tapahtui outoja asioita sen jälkeen kun muutin XYZ...

Ratkaisu: SpongeDocs ei ole tarpeaksi laaja kattaakseen kaikkia mahdollisia asioita mitä käyttäjät voivat tehdä että Sponge hajoaa. Kannataa aina ajatella mitä olit mahdollisesti tehnyt että palvelimesi ei enää toimi. Muistikortti saattaa olla hieman irti putsaamisen jälkeen, linkki saattaa osoittaa väärään paikkaan...

Spongessa on bugi¶

Ongelma: Mikään ylemmistä ei sovellu ongelmaan, ja Sponge ei silti toimi niin kuin pitäisi.

Solution: Time to get out the big guns. File a report on the SpongeForge or SpongeVanilla issue tracker, remembering to include details of the version of Forge and Sponge you are using, and a link to the relevant server log file.

Universumissa on jotakin vikaa¶

Valitettavasti emme voi auttaa sinua tässä. Olet nyt yksin.
