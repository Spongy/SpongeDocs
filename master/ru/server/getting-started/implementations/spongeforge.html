Установка SpongeForge¶

SpongeForge внедряется в Minecraft Forge, из-за чего Вы сможете запустить Forge-модификации. Вообще будет правильнее сказать, что Sponge это модификация для Forge, которая способна загружать Sponge-плагины, но это уже просто технические детали.

Пользователям, не хотящим использовать Minecraft Forge, лучше обратить внимание на SpongeVanilla.

Загрузка¶

Посмотрите на странице Загрузки.

Название загруженного файла¶

Когда вы скачаете SpongeForge название файла будет содержать некоторую важную информацию о версиях. Название содержит номер сборки Forge, с которой совместима эта версия SpongeForge. Другие сборки не будут работать, даже если они отличаются всего на несколько цифр.

Хотя обычно SpongeForge быстро обновляется до новой версии Forge, так что вам не стоит волноваться об актуальности версии Forge для работы со SpongeForge.

Формат названия файла такой: spongeforge-{MCVersion}-{ForgeVersion}-{SpongeAPIVersion}-{SpongeBuildId}

MCVersion The Minecraft version. Only clients compatible with this version can connect.
ForgeVersion

Версия Forge, для которой собран этот файл.
SpongeAPIVersion

Используемая этим файлом версия SpongeAPI. Это необходимо для Sponge плагинов.
SpongeBuildId

Номер сборки Sponge.

Пример¶

The file name spongeforge-1.8-1577-3.0.0-BETA-1000.jar is compatible with Minecraft version 1.8, requires build 1.8-11.14.4.1577 of Forge, provides SpongeAPI 3.0.0 and was build 1000 of SpongeForge.

Примечание

Normal Forge mods can usually run on any build of Forge for a given Minecraft version (e.g. 1.8.0) without any problems. However, SpongeForge needs to access, among other things, internal parts of Forge, which most mods shouldn’t be touching, let alone modifying as Sponge does. Since Forge is free to change internal code whenever they want to, its normal guarantee of backwards-compatibility doesn’t apply to SpongeForge.

Установка SpongeForge¶

Примечание

Если Вы используете (или планируете использовать) хостинг игровых серверов, то, возможно, там уже установлен Sponge.

Предупреждение

При использовании установщика Mojang, Mojang используют собственные версии Java, а не те, которые установлены в систему. В настоящее время установщик использует Java 1.8.0_25 для Windows и 1.8.0_60 для OSX. Обратите внимание, Sponge использует Java 1.8.0_40 или выше для нормального запуска. Вы можете скачать Лаунчер без встроенной Java здесь: официальный лаунчер Minecraft

Одиночная игра / Внутриигровой сервер¶

  1. Загрузите установщик Minecraft Forge с веб-сайта Minecraft Forge. Убедитесь в том, что Вы используете точно такой же номер сборки, который указан в названии скачанного файла Sponge.

  2. Установите Forge. Будет создан новый профиль Forge в лаунчере Minecraft.

  3. Откройте лаунчер Minecraft и выберите новый профиль Forge, который был ранее создан.

  4. Нажмите на “Options”, затем на “Open Game Dir”.

  5. Загрузите SpongeForge с сайта Sponge и переместите его в папку mods. Если папка отсутствует — создайте её.

  6. Sponge должен работать как и в одиночном режиме, так и если Вы откроете мир для сети.

Дальше узнайте, как Вы можете настроить Sponge и управлять сервером (в том числе установкой плагинов).

Выделенные серверы¶

Примечание

Если у Вас уже есть Forge-сервер, просто закиньте Sponge в папку mods. Не забудьте обновить версию Forge до той, что требует Sponge.

Установка Forge через командную строку¶

  1. Посетите сайт Minecraft Forge и кликните на “Show all downloads”, что просмотреть все доступные варианты. Найдите версию, соответствующую той, что указана в названии файла SpongeForge, и наведите на (i) после “Installer” для получения прямой ссылки на загрузку.

  2. Используйте любимый способ загрузки файла jar. Например: wget http://url.to/forge-version-installer.jar

  3. Из папки, где Вы хотите установить Forge, запустите скачанный jar с параметром``–installServer``. Например: java -jar forge-version-installer.jar --installServer

  4. Продолжение в “Добавление SpongeForge к Forge” ниже.

Установка Forge через GUI¶

  1. Загрузите установщик Minecraft Forge с веб-сайта Minecraft Forge для соответствующей версии, указанной в названии скачанного файла Sponge.

  2. Запустите установщик Forge, выберите пункт “Instal Server”, поменяйте папку на любую пустую папку для файлов сервера, затем нажмите OK.

  3. Продолжение в “Добавление SpongeForge к Forge” ниже.

Добавление SpongeForge к Forge¶

  1. Загрузите SpongeForge с сайта Sponge и переместите его в папку mods в каталоге вашего сервера. Если папка отсутствует — создайте её.

  2. Теперь Вы можете запустить сервер с помощью команды или запуска скрипта: java -jar forge-version-XYZ.jar.

  3. Если Вы запускаете сервер дома, настройте Перенаправление портов для того, чтобы другие смогли подключиться.

Дальше узнайте, как создать и использовать скрипты запуска, настроить Sponge и управлять сервером.
