Установка SpongeForge¶

SpongeForge внедряется в Minecraft Forge так что вы сможете запустить модификации для Forge. Вообще будет правильнее сказать, что Sponge это модификация для Forge, которая способна загружать плагины для Sponge, но это просто технические детали.

Пользователям, которым не требуется функционал Minecraft Forge, лучше обратить внимание на Установка SpongeVanilla.

Скачать¶

Посмотрите на странице Скачать.

Название загруженного файла¶

Когда вы скачаете SpongeForge название файла будет содержать некоторую важную информацию о версях. Оно содержит номер сборки Forge, с которой совместима эта версия SpongeForge. Другие сборки не будут работать, даже если они отличаются всего на несколько цифр.

Хотя обычно SpongeForge быстро обновляется до новой версии Forge, так что вам не стоит волноваться об актуальности версии Forge для работы со SpongeForge.

Формат названия файла будет таким: spongeforge-{MCVersion}-{ForgeVersion}-{SpongeAPIVersion}-{SpongeBuildId}

MCVersion

Версия Minecraft. Смогут подключиться только те клиенты, которые совместимы с этой версией.
ForgeVersion

Версия Forge, для которой собран этот файл. Предпочтительнее будет использовать конкретно эту версию Forge.
SpongeAPIVersion

Используемая этим файлом версия SpongeAPI. Это необходимо для Sponge плагинов.
SpongeBuildId

Номер сборки Sponge. Это вам потребуется предоставить при сообщении об ошибках или при обращении в поддержку.

Пример¶

Название файла spongeforge-1.8-1577-3.0.0-BETA-1000.jar говорит нам, что он совместим с версией Minecraft 1.8, требует сборку Forge 1.8-11.14.4.1577, обеспечивает SpongeAPI версии 3.0.0 и был собран SpongeForge 1000.

Примечание

Обычные моды Forge могут запускаться на любой сборке Forge для этой версии Minecraft (например 1.8.0) без каких-либо проблем. Однако SpongeForge, помимо всего прочего, требуется доступ во внутренние части Forge, которых не касается большинство модов, не говоря уже о модифицировании, как делает Sponge. Так как Forge может спокойно менять свой внутренний код когда захочет, то его нормальная гарантия обеспечения обратной совместимости не относится к SpongeForge.

Установка SpongeForge¶

Примечание

Если Вы используете панель управления игровыми серверами, то, возможно там уже есть Sponge.

Предупреждение

When using the Mojang installer, Mojang makes use of their own Java version and not the one you installed on your system. The installer currently ships with Java 1.8.0_25 for Windows and 1.8.0_60 for OSX. Note that Sponge requires at least 1.8.0_40 or above to run properly. You can grab the Launcher without included Java here: official Minecraft Launcher

Одиночная игра / Внутриигровой сервер¶

  1. Download the Minecraft Forge installer from the Minecraft Forge website. Make sure to use exactly the same build number as the one listed in the filename of the Sponge download.

  2. Установите Forge.

  3. Откройте лаунчер игры и выберите новый профиль.

  4. Нажмите на “Options” и нажмите на “Open Game Dir”.

  5. Загрузите SpongeForge с сайта Sponge и переместите его в папку mods. Если папка отсутствует — создайте её.

  6. Sponge должен работать как и в одиночном режиме, так и если вы откроете доступ к игре.

Next, learn how you can configure Sponge and how to manage your instance of Sponge (including installing plugins).

Выделенный сервер¶

Примечание

If you already have a Forge server, just put the Sponge mod into your mods folder. Remember to update your Forge version to match the one that sponge requires.

Установка Forge через командную строку¶

  1. Visit the Minecraft Forge website and click “Show all downloads” to view the full set of available options. Identify the version matching the one listed in the filename of the SpongeForge download, and hover over the (i) next to “Installer” to get the direct download link.
  2. Use your favorite download method to download the jar to its destination. Example: wget http://url.to/forge-version-installer.jar
  3. From the folder in which you wish to install Forge, execute the jar with the --installServer option. Example: java -jar forge-version-installer.jar --installServer
  4. Continue to Adding SpongeForge to Forge below.

Установка Forge через GUI¶

  1. Download the Minecraft Forge installer from the Minecraft Forge website for the version matching the one listed in the filename of the SpongeForge download.

  2. Запустите установщик Forge, выбирите любую пустую папку для сервера и нажмите OK.

  3. Continue to Adding SpongeForge to Forge below.

Adding SpongeForge to Forge¶

  1. Загрузите Sponge и установите его как обычный мод.

  2. Установка завершена.

  3. If operating from home, set up Перенаправление портов to ensure others can connect.
