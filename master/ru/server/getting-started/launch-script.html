Создание сценария (скрипта) запуска¶

Примечание

Эти инструкции подходят, если вы планируете запускать ваш Minecraft сервер на собственной машине. Большинство распространенный хостингов сделают это за вас.

Написание скрипта запуска¶

Сперва откройте текстовый редактор наподобие Atom, Sublime Text или же Notepad. Напишите (или вставьте) скрипт запуска для вашего сервера. Примеры простых скриптов для Windows, Mac OS X и Linux будут показаны ниже. Не забывайте про лимит ОЗУ вашей машины.

Примечание

Следующие примеры подходят для всех. Для серверов Forge, которые используют Sponge (coremod), замените forge-1.8-XYZ-universal.jar на то как файл Forge назван у вас в папке с сервером. Чтобы запустить SpongeVanilla сервер, замените forge-1.8-XYZ-universal.jar на название вашего SpongeVanilla.jar файла.

Windows¶

java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar
pause

Сохраните ваш скрипт запуска для Windows, как launch.bat.

Mac OS X¶

#!/bin/bash
cd "$(dirname "$0")"
java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar

Сохраните ваш скрипт запуска для Mac, как launch.command.

Linux¶

#!/bin/sh
cd "$(dirname "$(readlink -fn "$0")")"
java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar

Сохраните ваш скрипт запуска для Linux, как launch.sh.

Запуск скрипта запуска¶

Убедитесь что вы запускаете ваш скрипт в папке с сервером. Это для вашей безопасности. К сожалению, Sponge не сможет впитать ваши слезы если вы этого не сделайте.

Вы можете запустить ваш скрипт запуска двойным нажатием кнопкой мыши. Если вы используйте консоль или терминал, перейдите в папку с скриптом и запустите его. Не забывайте вы подтвердить Mojang EULA для того чтобы запустить сервер.

Примечание

По умолчанию, графическая консоль minecraft сервера отключена Sponge-ем, потому как она очень сильно нагружает процессор.

Предупреждение

Если вы получите ошибку про то что у вас нет прав, когда будете запускать ваш сервер на Mac, то попробуйте это:

  • Откройте терминал.

  • Введите chmod a+x, с пробелом в конце.

  • Перетащите скрипт запуска в терминал.

  • Нажмите клавишу Enter.
