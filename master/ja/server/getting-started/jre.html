Java のインストール¶

Java は Sponge や Minecraft の実行に必要です。おそらくすでに Java をインストールしているでしょうが、バージョンを更新する必要があるかもしれません。

SpongeにはJava 8 (具体的には 1.8.0_40 以上) が必要です。これより古いバージョンのJavaは使用されていませんし、Spongeは動作しません。バージョンの違いはメジャーバージョンのJava (6,7,8)が重視され、古いJavaではSpongeは動きません。

Java のインストール¶

Windows もしくは Mac OS X の場合は、 公式ウェブサイトからダウンロード できます。

LinuxユーザーはパッケージマネージャでOpenJDKをインストールしてください。OpenJDKはOracleのJavaをオープンソースにしたもので、ほぼ同様に動きます。ですが Linux用のOracleのJava も利用できます。しかし、パッケージマネージャでインストールされるほとんどのLinux用Java開発パッケージはOpenJDK用にインストールされることに注意してください。

32-bit と 64-bit¶

お使いのパソコンが対応しているなら、可能な限り64-bitのJavaを使用してください。Javaの公式ウェブサイトではお使いのパソコンが64-bitに対応しているかどうか検出する機能があります。

なぜなら64-bitのJavaはとても快調に動作し、3GB以上のRAM (メモリ) を使用できます。とにかく32-bitよりも64-bitがおすすめです。

現在の多くのコンピューターは 64-bit をサポートしています。

JDK と JRE¶

JRE (Java Runtime Environment) は Java アプリケーションの実行に使用されます。上記リンク先のダウンロードページでは、JREを提供しています。

JDK (Java Development Kit) はJavaアプリケーションの作成に使用します。Spongeプラグインの作成やSpongeの開発にJDKは必要ありません。しかし場合によってはSpongeなどの実行中のアプリケーションの診断のためにJDKが必要になることがあります。JDKは こちらのサイト からダウンロードできます。
