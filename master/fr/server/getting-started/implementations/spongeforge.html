Installer SpongeForge¶

SpongeForge intègre Minecraft Forge vous pouvez donc utiliser des mods Forge. En fait, c’est plutôt Sponge en lui-même qui est un mod Forge qui, ensuite, charge les plugins. Les explications s’arrêteront, la suite rentrant dans des détails techniques.

Les utilisateurs qui ne voudraient pas utiliser Forge devraient être intéressés par Installer SpongeVanilla.

Téléchargement¶

Jetez un œil à Téléchargement pour de plus amples informations.

Lire le nom du fichier en téléchargement¶

Quand vous téléchargez SpongeForge, le nom du fichier fournit des informations importantes concernant les version. Il inclut la version de Forge avec laquelle SpongeForge est compatible. Les autres versions, même très légèrement différentes ne sont pas supportées.

Cependant, SpongeForge est mis à jour assez vite après la sortie d’une nouvelle version de Forge. Ne vous en fait donc, vous n’aurez pas à utiliser de version obsolète de Forge pour utiliser SpongeForge.

Le format du nom de fichier est spongeforge-{MCVersion}-{ForgeVersion}-{SpongeAPIVersion}-{SpongeBuildId}

MCVersion

La version Minecraft. Seuls les clients compatibles avec cette version pourront se connecter.
ForgeVersion

La version de Forge pour laquelle ce fichier a été développé. Il est préférable que votre serveur utilise cette version.
SpongeAPIVersion

La version de l’API Sponge qu’implémente ce fichier. C’est ce dont dépendent les plugins Sponge.
SpongeBuildId

Le numéro de fabrication de Sponge. C’est ce que vous devez utilisez pour reporter des bugs ou chercher de l’aide sur le support.

Exemple¶

Le nom de fichier spongeforge-1.8-1577-3.0.0-BETA-1000.jar est compatible avec la version 1.8 de minecraft, a besoin de la version 1.8-11.14.4.1577 de Forge, contient l’API Sponge version 3.0.0 et est la version 1000 de SpongeForge.

Note

Les mods Forge normaux peuvent en principe fonctionner sur n’importe quelle version de Forge pour une version minecraft donnée (ex. 1.8.0). Cependant, SpongeForge a besoin d’accéder, entre autres, à des parties internes de Forge, qu’aucun mod ne devrait toucher, et encore moins modifier comme Sponge le fait. Puisque Forge est libre de changer son code interne autant qu’il le veut, il est normal qu’il n’y ait aucune garantie de rétro-compatibilité pour SpongeForge.

Installer SpongeForge¶

Note

Si vous utilisez (ou envisagez d’utiliser) un hébergeur pour votre serveur, ils peuvent avoir un panneau de contrôle qui peut installer Sponge pour vous.

Avertissement

Si vous utilisez l’installateur de Mojang, Mojang utilise sa propre version de java et n’utilise donc pas celle installée sur votre système. L’installateur fournit actuellement Java 1.8.0_25 pour Windows et 1.8.0_60 pour OSX. Notez que Sponge requiert au moins Java 1.8.0_40 ou supérieur pour fonctionner entièrement. Vous pouvez obtenir le Launcher sans version Java inclus ici: Launcher Minecraft Officiel

Solo / Serveur LAN en jeu¶

  1. Téléchargez l’installateur de Minecraft Forge depuis le site web Minecraft Forge. Soyez sûr d’utiliser exactement la même version que celle indiquée dans le nom du jar Sponge téléchargé.

  2. Exécutez le programme d’installation de Forge. Un nouveau profil Forge sera créé sur votre launcher Minecraft.

  3. Ouvrez le launcher de Minecraft, puis sélectionnez le nouveau profil Forge.

  4. Cliquez sur “Options” puis cliquez sur “Open Game Dir”.

  5. Téléchargez SpongeForge depuis le site de Sponge et mettez le dans le dossier mods. Créez le dossier si celui-ci n’existe pas.

  6. Sponge devrait fonctionner à la fois en solo et si vous ouvrez votre monde en LAN.

Ensuite, apprenez comment configurer Sponge et comment gérer votre instance de Sponge (installation de plugins compris).

Serveurs dédiés¶

Note

Si vous avez déjà un serveur Forge, mettez juste le mod Sponge dans votre dossier mods. Rappelez-vous de mettre à jour votre version de Forge pour correspondre à celle que Sponge requiert.

Installer Forge via la ligne de commande¶

  1. Visitez le site de Minecraft Forge et cliquez sur “Show all downloads” pour voir toutes les versions et options disponibles. Identifiez la version inscrite dans le nom du jar SpongeForge téléchargé, et survolez le (i) à côté de “Installer” pour avoir le lien de téléchargement direct.

  2. Utiliser votre manière de téléchargement favorite pour obtenir le jar à sa destination. Exemple: wget http://url.to/forge-version-installer.jar

  3. Depuis le fichier dans lequel vous voulez installer Forge, exécutez le jar avec l’option --installServer. Exemple: java -jar forge-version-installer.jar --installServer

  4. Continuez à ajouter SpongeForge à Forge ci-dessous.

Installer Forge via une interface (GUI)¶

  1. Téléchargez l’installateur Minecraft Forge depuis le site de Minecraft Forge pour avoir la version correspondant à celle inscrite dans le nom du fichier SpongeForge téléchargé.

  2. Exécutez l’installateur de Forge, sélectionnez “Install Server”, choisissez un dossier vide pour placer les fichiers du serveur, puis cliquez sur OK.

  3. Continuez à ajouter SpongeForge à Forge ci-dessous.

Ajouter SpongeForge à Forge¶

  1. Téléchargez SpongeForge depuis le site de Sponge et mettez le dans le dossier mods de votre serveur. Créez le dossier s’il n’existe pas encore.

  2. Vous pouvez maintenant lancer le serveur via le script de commande ou de lancement java -jar forge-version-XYZ.jar.

  3. Si vous faites cela chez vous, configurez le port-forward pour vous assurer que les autres puissent se connecter.

Ensuite, apprenez comment créer et utiliser un launch-script, configurer Sponge et gérer cotre serveur (installation de plugins inclut).
