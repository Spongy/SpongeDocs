Installation de Java¶

Java est requis pour faire fonctionner Sponge et Minecraft. Vous avez très probablement déjà Java, mais il se peut que vous deviez le mettre à jour.

Sponge requiers Java 8 (il est conseillé d’avoir la version 1.8.0_40 ou supérieur), les anciennes versions de Java ne sont plus supportées et ne fonctionneront donc pas avec Sponge. La différence entre les versions majeures de Java (6, 7, 8) est très significative, les anciennes versions ne sont donc pas capables de pouvoir faire fonctionner Sponge correctement.

Installation de Java¶

Si vous avez Windows ou Mac OS X d’installé sur votre ordinateur, vous pouvez télécharger Java depuis le site officiel.

Les utilisateurs de Linux peuvent installer OpenJDK via leur gestionnaire de paquets. OpenJDK est la version open source de la version de Java d’Oracle, et cela devrait fonctionner tout aussi bien. Cependant, il est aussi possible de télécharger la version d’Oracle pour Linux, mais sachez que de nombreux paquets Linux dépendants de java seront de toute façon toujours installer avec OpenJDK.

32-bit vs. 64-bit¶

Si votre ordinateur le supporte, vous devriez utiliser la version 64-bit de Java lorsque possible. L’installateur Java du site de Java devrait détecter si votre ordinateur est prêt pour du 64-bit.

Parce que la version 64 bits de Java fonctionne beaucoup mieux, et vous permet également à Java d’utiliser plus de ~ 3 Go de votre RAM (mémoire), nous vous la recommandons de toujours au dessus de 32-bit.

La plus part des ordinateurs récent supportent le 64-bit.

JDK vs. JRE¶

Le JRE (Java Runtime Environnent) est utilisé pour exécuter des applications Java. La page de téléchargement ci-dessus fournit le JRE.

Le JDK (Java Development Kit) est utilisé pour creer des applications Java, et vous n’en n’avez pas besoin du moment que vous souhaitez créer des plugins pour Sponge ou travailler sur Sponge. Vous télécharger le JDK ici.
