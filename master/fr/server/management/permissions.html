Gestion des permissions¶

Vous pouvez configurer qui peut accéder à quelles commandes sur votre serveur en utilisant les permissions. Les permissions spécifiques aux commandes de Sponge, Forge et Minecraft sont répertoriées sur la page des Commandes.

Niveau de l’opérateur¶

Minecraft possède une méthode simple pour donner des permissions: donner le statut d’opérateur à un joueur (ou “op” pour faire court). Les informations sur le statut d’op peuvent êtres trouvées à cette adresse: http://minecraft.gamepedia.com/Op

Les capacités des permissions des OP peuvent être réglées en modifiant le paramètre op-permission-level dans le fichier server.properties.

Une liste de toutes les commandes natives des serveurs Minecraft disponibles aux joueurs OP peut être trouvée sur le Minecraft Wiki.

Avertissement

Minecraft ne dispose pas de permissions fines, seulement le statut d’op. C’est un niveau très élevé d’autorisation qui ne doit être réservé pour les joueurs de confiance. La configurations de permissions plus complexes nécessitent l’utilisation d’un plugin ou d’un mod. Sponge n’est pas un plugin de gestion des autorisations.

Note

Quelques plugins et mods peuvent donner certaines permissions aux OP.
