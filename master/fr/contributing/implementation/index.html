Développer Sponge¶

This sections describes the best way to approach the project when you’re intending to develop Sponge itself. It is mandatory to be familiar with git and our Style de Code. Basic knowledge of the La Structure du Sponge Project is also very helpful.

Si vous voulez nous rejoindre dans la création de Sponge, dirigez-vous tout droit vers la page Soumettre une Contribution.
