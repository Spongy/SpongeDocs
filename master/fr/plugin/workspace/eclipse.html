Préparation d’Eclipse¶

This article describes how to configure your Eclipse workspace for plugin development with SpongeAPI and a build system such as Maven or Gradle.

Note

A tutorial by Mumfrey showing the setup of a Sponge workspace in Eclipse, using the new features in Buildship and ForgeGradle, can be viewed Here.

Gradle¶

Creating your project¶

Vous devez d’abord installer Gradle Integration Plugin avant d’utiliser Gradle dans Eclipse. Cela ne doit être fait qu’à la création de votre premier projet.

Note

En général, vous n’avez pas besoin du module optionnel Spring, distribué avec ce plugin. Vous pouvez donc le décocher pendant l’installation.

  • Ouvrez Eclipse.

  • Cliquez sur Help > Eclipse Marketplace.

  • Cherchez le Gradle Integration Plugin.

  • Installez le plugin Gradle Integration Plugin.

Vous devez ensuite créer votre projet.

  • Cliquez sur File > New > Other.

  • Sélectionnez Gradle > Gradle Project.

  • Cliquez sur Next.

  • Choisissez un nom de projet, puis cliquez sur Terminé.

Editing the build script¶

  • Ouvrez build.gradle dans le navigateur.

  • Edit the build script according to the instructions at Configurer Gradle.

  • Faites un clique-droit sur votre projet, ensuite sélectionnez Gradle > Refresh Dependencies.

Importing your project¶

If you’ve already started with your project and want to import it again at a later point you need to import it instead of re-creating it inside your IDE:

  • Click File > Import
  • Select Gradle > Gradle Project
  • Navigate to the root folder of the project
  • Click Finish

Maven¶

Creating your project¶

  • Ouvrez Eclipse.

  • Cliquez sur File > New > Other.

  • Selectionnez Maven sur le côté gauche du menu contextuel, puis sélectionnez Maven Project sur le menu de droite.

  • Cliquez sur Next.

  • Sélectionnez Create a simple project, tant que vous n’avez pas besoin d’une configuration plus avancée.

  • Entrez votre Group ID, Artifact ID, Version, votre nom de projet et votre description du projet.

    • Your Group ID should usually correspond to your Java package name. See Créer la Classe de Votre Plugin for details.
    • Your Artifact ID should usually correspond to your plugin ID you chose earlier, e.g. myplugin.

    • Libre à vous de choisir votre Version.

  • Cliquez sur Finish.

Astuce

Souvent Éclipse n’ouvre pas nouveaux projets après leur création. Dans ce cas, essayez de fermer l’écran de bienvenue d’Eclipse .Votre projet devrait être ouvert derrière celle-ci.

Editing the project configuration¶

  • Ouvrez pom.xml dans le navigateur.

  • Edit the build configuration according to the instructions at Configurer Maven.
  • Refresh your Maven project.

Importing your project¶

If you’ve already started with your project and want to import it again at a later point you need to import it instead of re-creating it inside your IDE:

  • Click File > Import
  • Select Maven > Existing Maven Projects
  • Navigate to the root folder of the project
  • Click Finish
