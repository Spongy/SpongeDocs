Travailler avec du Texte¶

Formatted Text can be created using the Text.Builder, as described in this section. The robust Text API can be used in a variety of ways to combine styling, coloring, and Text actions.

La pagination, un processus de séparation du contenu (tels que les listes de texte) en plusieurs pages, sera aussi abordée dans cette partie.
