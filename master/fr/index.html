Documentation de Sponge¶

Introduction¶

Bienvenue sur SpongeDocs, la documentation officielle pour le projet open source Sponge.

La SpongeAPI est une toute nouvelle API de plugins pour Minecraft. Vous pouvez l’utiliser sur un serveur entièrement vanilla (avec SpongeVanilla) ou sur un serveur MinecraftForge entièrement moddé (avec SpongeForge). Prenez une copie ici.

Vous ne savez pas trop ce que vous cherchez? Regardez la section À propos du projet Sponge, qui contient une courte Introduction, notre Foire aux Questions ainsi qu’un article à propos de La Structure du Sponge Project.

Les propriétaires de serveur devraient lire nos conseils et instructions dans Créer un serveur sur la manière d’installer, d’utiliser et de maintenir un serveur basé sur Sponge.

Le projet Sponge possède plusieurs composants principaux. Vous pouvez en apprendre plus sur la structure du projet ici ou visiter leurs dépôts sur GitHub:

SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | SpongeHome | Mixin | Ore

Note

Sponge est toujours en développement. Les versions bêta de SpongeForge et SpongeVanilla sont disponibles sur la page des Téléchargement, mais une version officielle ne sera disponible que plus tard durant l’année 2016.
