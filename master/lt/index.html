Sponge dokumentacija¶

Įvadas¶

Welcome to SpongeDocs, the official documentation for the open-source Sponge project.

The SpongeAPI is your all new plugin API for Minecraft. You can run it on a plain vanilla server (as SpongeVanilla) or on your fully modded MinecraftForge server (as SpongeForge). Grab your copy here.

Not sure what you’re looking for? Try our Apie Sponge projektą section, which contains a short Įvadas, our Dažniausiai užduodami klausimai and an article about Sponge projekto struktūra.

Server owners should read our tips and instructions in Serverio kūrimas on how to install, run and maintain a Sponge based server.

The Sponge project has several main components. Learn more about the project structure here or visit their repositories on GitHub:

SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | SpongeHome | Mixin | Ore

Pastaba

Sponge is still in development. Beta builds of both SpongeForge and SpongeVanilla are currently available for Siuntiniai, but an official release will not be available until later in 2016.
