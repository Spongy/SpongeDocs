Grundlegende Konzepte¶

Die ‘Sponge Economy API’ hat ein paar grundlegende Komponenten, mit denen Entwickler vertraut sein sollten:

  • EconomyService

  • Währung

  • Konten

  • Transactions

Der ‘EconomyService’¶

Der ‘EconomyService’ ist die Grundlage der ‘Economy API’. Es wird gebraucht, um über die ‘Economy API’ zu interagieren, speichert die Währungen einer ‘Economy’ und stellt Methoden für die Kontenverwaltung bereit. Die ‘EconomyService’ JavaDocs können hier gefunden werden: <https://jd.spongepowered.org/index.html?org/spongepowered/api/service/economy/EconomyService.html>

Warnung

Sponge stellt keine Standardimplementierung für den ‘EconomyService’ bereit. Es liegt an den Plugins selbst, die ‘Economy API’ zu implementieren. Das heißt, dass du ein anderes Plugin brauchst, das diesen Service implementiert, oder dass du die ‘Economy API’ selbst implementieren musst, wenn du von ihr Gebrauch machen möchtest.

Währung¶

The org.spongepowered.api.service.economy.Currency object represents a form of Currency. Currency stores a display name (plural and singular), a symbol, the amount of fractional digits, and whether the currency is the default currency for the economy. If the economy plugin chooses, it can support multiple currencies.

Konten¶

Accounts are used to store economy information about a specific player or other object (i.e. bank, business, entity). There are two account types in the Sponge Economy API:

Virtual Accounts¶

Virtual accounts are tied to an identifier, which is stored as a string. Virtual accounts can be tied to almost anything, but are commonly used for things such as banks, or non-players. To get the id of a virtual account, use getIdentifier().

Unique Accounts¶

Unique accounts are tied to a UUID, usually a player. To get the UUID of an unique account, use getUUID().

Transactions¶

Transactions represent an account’s change in balance. There are currently three types of transactions:

  • Deposit: Occurs when an account has funds added to it
  • Withdraw: Occurs when an account has funds removed from it
  • Transfer: Occurs when an account exchanges funds with another account

When a transfer occurs, the EconomyTransactionEvent is fired. Using this event, you can get the TransactionResult. The TransactionResult stores data about the transaction that occurred, including:

  • Account involved
  • Currency involved
  • Amount of currency involved
  • Transaction type
  • Result of the transaction

You can view all possible transaction results at the Sponge JavaDocs.
