Git installieren¶

Git ist ein quelloffenes (englisch open-source) Versionierungssystem, welches die gleichzeitige Arbeit an Projektes deutlich vereinfacht.

Das Sponge Projekt verwendet, so wie tausende andere Open Source Projekte auch, Git Repositories von GitHub. Somit ist Git ein entscheidendes Werkzeug bei der Entwicklung von Sponge und der Sponge API.

Die Git Internetseite enthält reichlich Dokumentation, ihre Downloadseite beinhaltet eine Reihe von Optionen für GUI clients, geeignet für verschiedene Betriebssysteme.

Download¶

Windows¶

GitHub Desktop ist eine einfache Methode Git auf Windows zu installieren, da Git ein Teil der Software ist.

Nach der Installation sollte der Computer neu gestartet werden.

Max¶

Es gibt mehrere Möglichkeiten Git auf Mac OS X zu installieren.

Die einfachste Methode ist das Installieren der Xcode Command Line Tools.

Warnung

Diese Anweisungen funktionieren nicht auf OSX Versionen, die älter als Mavericks sind. Wenn du eine ältere Version als Mavericks einsetzt, instalieren stattdessen den Github Client.

  1. Öffne das Terminal.

  2. Führe xcode-select --install aus.

  3. Installiere es und schnapp dir einen Keks oder Kaffee, während du wartest.

  4. Führe git im Terminal aus.

Alternative kannst du auch GitHub Desktop installieren. Git ist Teil von GitHub Desktop.

Nach der Installation sollte der Computer neu gestartet werden.

Linux und Unix¶

Am einfachsten lässt sich Git über den mitgelieferten Paketmanager von Linux installieren.

Bemerkung

Eventuell müssen den folgenden Befehlen ein “sudo” vorangestellt werden.

  1. Öffne das Terminal.

  2. Starte apt-get install git wenn du mit einer Ubuntu oder Debian-basierten Distribution arbeitest. Falls du Fedora nutzt, führe yum install git aus.

GitHub Desktop ist derzeit nicht für Linux verfügbar.

Nach der Installation sollte der Computer neu gestartet werden.

Setup¶

Wer bist du?¶

Stelle sicher, dass Git richtig konfiguriert und deine Identität richtig hinterlegt ist, bevor du dich in die Arbeit stürzt. Öffne deine Kommandozeile und geben ein:

git config --list

Look for user.name and user.email. If they are not the same username and email as your GitHub account then set them:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Tipp

Trage nichts zum Repository von Sponge bei, bis du deinen Benutzernamen und deine E-Mail-Adresse richtig eingetragen hast.
