Installieren einer IDE¶

Was ist eine IDE?¶

Die Installation von einer IDE - auch bekannt als eine integrierte Entwicklungsumgebung - wird empfohlen für die Entwicklung von Sponge-Plugins und/oder Arbeiten mit Sponge selbst. Eine IDE vervollständigt automatisch Namen und informiert Sie über Fehler, z. B. nicht übereinstimmende geschweifte Klammern, in Ihrem Code.

Welche IDE sollte ich wählen?¶

Jeder IDE hat seine vor- und Nachteile, und es gibt keine einzelne beste IDE für die Sponge-Entwicklung. Das Aussuchen einer IDE ist weitgehend eine Frage des persönlichen Geschmacks. Davon abgesehen, ist es erwähnenswert, dass die `IntelliJ IDEA Community Edition < https://www.jetbrains.com/idea/download >`__ ziemlich beliebt bei einer Mehrheit des Kernteams von Sponge ist.
