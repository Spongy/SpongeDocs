SpongeForge installieren¶

SpongeForge basiert auf Minecraft Forge. Das heißt, dass es auch möglich ist Forge-Mods zu installieren. In Wirklichkeit ist es so, dass Sponge selber eine Forge-Mod ist, welche die Sponge-Plugins lädt. Das ist aber nur ein technisches Detail.

Nutzer, die nicht Minecraft Forge verwenden möchten, sollten sich SpongeVanilla ansehen.

Download¶

Sehe unter Download für weitere Informationen nach.

Die Bedeutung des Dateinamens¶

Wenn du SpongeForge herunterlädst, wird der DAteiname wichtige Versionsinformationen enthalten. Er enthält die Forge Buildnummer, welche die Version zeigt, mit der SpongeForge kompatibel ist. Andere Builds von Forge, auch wenn sie nur weniger Buildnummern abweichen, werden nicht von uns unterstützt.

Normalerweise wird SpongeForge sehr rasch auf neue Forge Versionen angepasst, also musst du dich nicht darum sorgen eine veraltete version von Forge einsetzen zu müssen.

Das Format des Dateinamens ist spongeforge-{MCVersion}-{ForgeVersion}-{SpongeAPIVersion}{SpongeBuildId}

MCVersion

Die Minecraft-Version. Es können sich nur Clients, die mit dieser Version kompatibel sind verbinden.
ForgeVersion

Die Version von Forge, für die der Sponge (coremod) erstellt wurde. Vorzugsweise nutzt du genau diese Forge-Version.
SpongeAPIVersion

Die Version der Sponge-API, die von diesem Coremod unterstützt wird. Hiervon hängt die korrekte Funktion der Sponge Plugins ab.
SpongeBuildId

Die Versionsnummer von Sponge. Sie sollte angegeben werden, wenn du einen Fehler meldest oder nach Unterstützung bei der Fehlerbehebung fragst.

Beispiel¶

Der Dateiname spongeforge-1.8-1577-3.0.0-BETA-1000.jar zeigt, das SpongeForge kompatibel mit Minecraft version 1.8 ist, Forgebuild 1.8-11.14.4.1577 benötigt, SpongeAPI 3.0.0 bereitstellt and Build 1000 von SpongeForge ist.

Bemerkung

Normalerweise können Forgemods auf jedem Build von Forge für eine Minecraft Version (z.B. 1.8.0) ohne Probleme laufen. Allerdings braucht SpongeForge Zugang zu internen Teilen von Forge, welche die meisten Mods nicht nutzen oder gar verändern, wie Sponge es tut. Da Forge den internen Code ändern kann, wie es nötig ist, trifft diese Kompatibilität nicht auf SpongeForge zu.

SpongeForge installieren¶

Bemerkung

Wenn du einen Spiel-Hoster verwendest, hast du vielleicht im Control Panel die Möglichkeit, Sponge zu installieren.

Warnung

Wenn du den Mojang-Installer nutzt, wird automatisch eine separate Java-Version auf deinem System installiert. Der Installer enthält derzeit Java 1.8.0_25 für Windows und Java 1.8.0_60 für OSX. Beachte das Sponge mindestens Java 1.8.0_40 oder neuer benötigt um zu laufen. Daher musst du unter Windows den offiziellen Launcher ohne Java einsetzen: official Minecraft Launcher

Einzelspieler / In-Game-LAN-Server¶

  1. Lade den Minecraft Forge-Installer von der Minecraft Forge-Webseite herunter. Stelle sicher, das du exakt die :doc:`im Dateinamen des Sponge-Downloads genannte Forge Version <spongeforge>`einsetzt.

  2. Starte den Forge Installer, welcher ein neues Profil in deinem Minecraft-Launcher erstellen wird.

  3. Öffne den Minecraft-Launcher und wähle das Forge-Profil aus.

  4. Klicke auf “Optionen” und anschließend auf “Open Game Dir”.

  5. Lade dir SpongeForge von der Sponge Webseite herunter und verschiebe die Datei in den Ordner namens “mods”. Erstelle ihn, falls er nicht existiert.

  6. Sponge funktioniert im Singleplayer und auch, wenn du deine Welt im LAN verfügbar machst.

Als Nächstes lerne wie man Sponge konfigurieren kann und wie man seinen Server verwaltet (beinhaltet das Installieren von Plugins).

Dedizierte Server¶

Bemerkung

If you already have a Forge server, just put the Sponge mod into your mods folder. Remember to update your Forge version to match the one that Sponge requires.

Forge über die Kommandozeile installieren¶

  1. Besuche die Minecraft Forge Seite und klicke auf “Show all downloads” um alle verfügbaren Versionen anzuzeigen. Suche dir die passende Version heraus, die im Dateinamen von SpongeForge angegeben ist und fahre über das (i) neben “Installer” um den direkten Downloadlink zu bekommen.

  2. Nutze deine übliche Downloadmethode um die jar Datei herunterzuladen. Beispiel: wget http://url.to/forge-version-installer.jar

  3. Führe im Ordner in den du Forge installieren möchtest die jar Datei mit der --installServer Option aus. Beispiel: java -jar forge-version-installer.jar --installServer

  4. Fahre mit dem Hinzufügen von SpongeForge zu Forge fort.

Forge mit der GUI installieren¶

  1. Lade den Minecraft Forge-Installer von der Minecraft Forge-Webseite herunter. Wähle die Version, die im Dateinamen von SpongeForge genannt wird.

  2. Führe den Forge-Installer aus und klicke auf “Install Server”, wähle einen leeren Ordner aus, in dem die Server-Dateien platziert werden sollen, und klicke anschließend auf OK.

  3. Fahre mit dem Hinzufügen von SpongeForge zu Forge fort.

SpongeForge zu Forge hinzufügen¶

  1. Lade die SpongeForge von der Sponge Homepage herunter und lege sie in den Ordner mods in deinem Serververzeichnis ab. Erstelle das Verzeichnis falls es noch nicht existiert.

  2. Jetzt kannst du den Server mit dem Startscript oder dem Startbefehl java -jar forge-version-XYZ.jar starten.

  3. Falls dein Server hinter einer Firewall steht, schau dir Portweiterleitung an, um sicherzustellen das sich andere Spieler verbinden können.
