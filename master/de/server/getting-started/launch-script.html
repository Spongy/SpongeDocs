Ein Start-Skript erstellen¶

Bemerkung

Diese Anleitung wird nur benötigt, wenn du einen Minecraft-Server auf deinem eigenen PC betrieben möchtest. Die meisten Minecraft-Server-Anbieter erstellen das Start-Skript automatisch.

Ein Start-Skript schreiben¶

Öffne als erstes einen Texteditor wie Atom, Sublime oder Notepad. Schreibe dein Start-Skript oder füge es ein die Datei ein. Beispiele für Windows, Mac OS X und Linux sind unten zu finden. Beachte die RAM-Grenze deines PCs.

Bemerkung

Die folgenden Beispiele sind allgemein. Um einen Forge-Server, der den Sponge-Coremod verwendet, zu starten, ändere forge-1.8-XYZ-universal.jar in die entsprechende Version, die sich in deinem Serververzeichnis befindet. Um einen SpongeVanilla-Server zu starten ändere den Dateiname forge-1.8-XYZ-universal.jar in den Namen SpongeVanilla.jar.

Windows¶

java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar
pause

Speicher dein Windows-Start-Skript als launch.bat.

Max OS X¶

#!/bin/bash
cd "$(dirname "$0")"
java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar

Speicher dein Mac OS X-Start-Skript als launch.command.

Linux¶

#!/bin/sh
cd "$(dirname "$(readlink -fn "$0")")"
java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar

Speicher dein Linux-Start-Skript als launch.sh.

Ein Start-Skript ausführen¶

Stelle sicher, dass du dein Start-Skript aus einem Ordner, den du für den Server angelegt hast, ausführst. Dies ist nur für dich. Spongie kann deine Tränen nicht trocknen, wenn du diesen Rat nicht befolgst.

Du kannst das Start-Skript durch einen Doppelklick starten. Falls du eine Konsole oder ein Terminal einsetzt, navigiere zum Serverordner und führe es dort aus. Beachte, dass du der Mojang-EULA zustimmen musst, um einen Server ausführen zu können.

Bemerkung

The default Minecraft server GUI console is disabled by Sponge, because it is very processor-intensive.

Warnung

Wenn du bei dem Versuch deinen Server auf OS X zu starten einen Berechtigungsfehler erhälst, versuche Folgendes:

  • Öffne das Terminal.

  • Gib chmod a+x mit einem Leerzeichen am Ende ein.

  • Zieh dein Start-Skript in das Terminal.

  • Drücke die Eingabetaste.
