Installieren von Plugins¶

Was sind Plugins?¶

Plugins sind Dateien, geschrieben in Java, die die Art wie der Server funktioniert verändern. Dies geschiet hauptsächlich um den Server um Funktionen zu erweitern. Sie erlauben Server-Besitzern Modifkationen vorzunehmen, beispielsweise durch Hinzufügen eines Wirtschaftssystems, Managen von Teleportern und Berechtigungen, etc.

Finden von Plugins¶

Warnung

Lade Plugins nur von vertrauenswürdigen Quellen herunter! Schädliche Plugins können verwendet werden, um Fremden unerlaubten Zugriff auf deinen Server oder Computer zu ermöglichen.

Installation¶

SpongeForge¶

Um Plugins zu installieren, füge diese in den “mods”-Ordner von deinem Spiel- bzw. Serververzeichnis ein.

Wenn dein Download im .zip-Format vorliegt, musst ihn zuerst entpacken, um die dort enthaltene .jar-Datei zu finden.

SpongeVanilla¶

Bemerkung

Der Einheitlichkeit halber zwischen dem SpongeForge und SpongeVanilla werden Plugins im “mods”-Ordner hinterlegt, wenn du einen SpongeVanilla-Server betreibst.

Um Plugins zu installieren, füge diese in den “mods”-Ordner von deinem Spiel- bzw. Serververzeichnis ein.

Wenn dein Download im .zip-Format vorliegt, musst ihn zuerst entpacken, um die dort enthaltene .jar-Datei zu finden.

Häufige Probleme¶

Wenn du Probleme mit einem Plugin hast, prüfe folgendes:

  • Ist das Plugin mit deiner Minecraft Version kompatibel? Sponge versucht alte Plugins funktionsfähig zu halten, dies ist aber nicht immer möglich. In den meisten Fällen funktionieren Plugins die auf einem stabilen Release basieren jedoch weiterhin, ohne für die neue Version angepasst werden zu müssen.

  • Nutzt du die aktuelle Java Version? Sponge wurde für Java 8 entwickelt und ist bekannt dafür, einen Bug in JRE Versionen älter als 1.8.0_40 auszulösen. Stelle also sicher das deine Java Version neuer ist.

  • Eventuell ist das Plugin veraltet. Existiert eine neuere Version des Plugins?

  • Braucht das Plugin eine bestimmte Implementierung oder Version von Sponge? Einige Plugins können die Sponge-API umgehen oder anderweitig auf Details zugreifen, die sich je nach Minecraft Version oder Platform verändern. Prüfe dies auf der Seite, wo du das Plugin heruntergeladen hast.
