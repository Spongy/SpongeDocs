Documentazione di Sponge¶

Introduzione¶

Benvenuti su SpongeDocs, la documentazione ufficiale per il progetto open-source `Sponge <http://spongepowered.org>.

La Sponge API è la tua nuova plugin API per Minecraft. Puoi eseguirla su un server vanilla (come SpongeVanilla) o sul tuo server MinecraftForge completamente moddato (come SpongeForge). Ottieni la tua copia qui.

Non sei sicuro di quello che stai cercando? Prova la sezione A proposito del progetto Sponge..., che contiene una breve Introduction, le nostre Domande frequenti e un articolo sulla The Structure of the Sponge Project.

I proprietari di server dovrebbero leggere i nostri Creating a Server su come installare, eseguire e mantenere un server basato su Sponge.

Il progetto Sponge ha attualmente diverse componenti, visita le loro repository su GitHub o ottieni più informazioni qui: SpongeForge | SpongeVanilla | SpongeAPI | SpongeDocs | Mixin | Ore

Nota

Sponge è ancora in sviluppo. Sono attualmente disponibili delle Beta Build di SpongeForge e Alfa build di SpongeVanilla attraverso la pagina Download, ma un rilascio ufficiale non sarà disponibile fino al tardo 2016.
