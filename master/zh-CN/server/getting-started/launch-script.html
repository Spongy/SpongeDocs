创建启动脚本¶

注解

本文面向那些打算用自己的机器来跑Minecraft服务器的人，大部分Minecraft服务器供应商会帮你做好这部分。

写一个启动脚本¶

首先打开文本编辑器，比如Atom，Sublime Text或者记事本(推荐使用Notepad++)，然后写下(或者粘贴)用于你服务器的启动脚本。对于Windows，MacOS还有Linux系统的范例在下面，但还是请谨记你的机器的内存限制。

注解

下面的例子是通用的，对于Forge服务器，将文中的 forge-1.8-XYZ-universal.jar 换成你使用的Forge版本。对于原版Sponge服务器，将 forge-1.8-XYZ-universal.jar 命名为SpongeVanilla.jar文件的名字。

Windows¶

java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar
pause

将你的Windows脚本保存为 launch.bat.

Mac OS X系统¶

#!/bin/bash
cd "$(dirname "$0")"
java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar

将你的Mac启动脚本保存为 launch.command.

Linux系统¶

#!/bin/sh
cd "$(dirname "$(readlink -fn "$0")")"
java -Xms1G -Xmx2G -jar forge-1.8-XYZ-universal.jar

将你的Linux启动脚本保存为 launch.sh.

打开启动脚本¶

请确保你的启动脚本放置于你的服务器文件夹之外，这是为了你好，很不幸的是，如果你不这么做，Sponge也没有办法吸干你的眼泪。

您可以通过双击它来运行你的启动脚本。如果你正在使用一个控制台或终端，导航到该脚本的目录并运行它。请记住，为了运行一个服务器，您必须同意 Mojang EULA 。

注解

海绵服务器禁用minecraft默认服务器GUI，因为它是处理加强

警告

当你尝试启动在 Mac 上的服务器时，出现权限错误，试试这个:

  • 打开终端.

  • 在结尾处空白处键入 chmod a+x

  • 将你的启动脚本拖到终端。

  • 按 enter 键。
