安装Git¶

Git是一个开源的版本控制系统，它极大的方便了开发人员间的合作。

Sponge和数以千计的开源项目一样，将它的代码托管在GitHub上。因此，Git是Sponge及API开发中至关重要的工具。

Git网站 有大量的文档，和他们下载页面提供一系列的 GUI 客户端适用于各种操作系统。

下载¶

Windows系统¶

GitHub Desktop 是一种在 Windows 上简便的安装式Git，因为 Git 是包括作为软件的一部分。

我们建议在安装完成后重启计算机。

Mac¶

有多种方式在Mac OS X上安装Git。

最简单的方法是安装Xcode Command Line Tools。

警告

下列内容不适用与比Mavericks更老的系统。如果你正在使用一个旧的系统，请安装Github Client。

  1. 启动终端。

  2. 执行 xcode-select --install.

  3. 等待安装，也许你可以乘机来块饼干。

  4. 在终端运行 git

或者，你可以安装 GitHub Desktop. Git 是可用作为 GitHub 桌面安装的一部分。

我们建议在安装完成后重启计算机。

Linux和Unix¶

在Linux上安装git的最简单方法是使用你的发行版的包管理系统。

注解

你也许需要在下列命令前加上 sudo

  1. 启动终端。

  2. 如果你使用Ubuntu之类的Debian系系统，执行 apt-get install git。如果是Fedora，执行 yun install git。

GitHub 桌面是目前 Linux不可用，不像 Windows 和 mac。

我们建议在安装完成后重启计算机。

安装¶

你是谁？¶

在开始使用 git 和存储库之前，请确保您的 git 配置已设置了您的身份。打开你的 CLI 并输入:

git config --list

寻找 user.name 和 user.email。如果他们不是您的 Github 帐户相同的用户名和电子邮件请设置它们:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

小技巧

在设定完成你的 user.name 和 ``user.email``之前，不要对Sponge仓库做任何提交。
