Git Installeren¶

Git is een open-source versie controle systeem en helpt enorm bij de ontwikkeling van collaboratieve projecten.

Het Sponge project host, net als duizenden andere open-source projecten, zijn Git repositories op GitHub. Git is dus van cruciaal belang voor de ontwikkeling van Sponge en de Sponge API.

De Git website heeft aanzienlijke documentatie en hun downloadpagina biedt een scala aan mogelijkheden voor GUI clients geschikt voor verschillende besturingssystemen.

Download¶

Windows¶

GitHub Desktop is een makkelijke manier om Git te installeren op Windows, aangezien Git een deel van de software is.

Het herstarten van je computer na het installeren van Git is aangeraden.

Mac¶

Er zijn enkele manieren om Git te installeren op Mac OS X.

The makkelijkste methode is het installeren van Xcode Command Line Tools.

Waarschuwing

Deze instructies werken niet op Macs die een versie ouder dan Mavericks draaien. Als je zo’n versie van OS X draait, installeer dan de GitHub client.

  1. Start de terminal.

  2. Voer xcode-select --install uit.

  3. Installeer het en neem een koekje tijdens het wachten.

  4. Voer git uit via de Terminal.

Als alternatief kunt u ook GitHub Desktop installeren. Git is beschikbaar als deel van de GitHub Desktop installatie.

Het herstarten van je computer na het installeren van Git is aangeraden.

Linux and Unix¶

De simpelste methode om Git te installeren op Linux is door je package manager te gebruiken die bij je Linux distributie hoort.

Notitie

You moet mogelijk de commando’s laten voorafgaan door sudo.

  1. Start de terminal.

  2. Voer apt-get install git uit als je op een Ubuntu of Debian-gebaseerde distributie werkt. Voer yum install git uit als je Fedora gebruikt.

GitHub Desktop is op het moment niet beschikbaar voor Linux, maar wel voor Windows en Mac.

Het herstarten van je computer na het installeren van Git is aangeraden.

Set-up¶

Wie bent u?¶

Voordat je begint te werken met Git en de repository moet je controleren dat de je identiteit correct is ingesteld in de Git configuratie. Open uw CLI en enter:

git config --list

Zoek naar user.name en user.email. Stel dat dit niet dezelfde gebruikersnaam en e-mail zijn als uw GitHub account stel ze dan in:

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Tip

Ga niet verder met bijdragen tot de Sponge repository tenzij je user.name en user.email correct zijn ingesteld.
